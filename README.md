# Teaching Apps

A collection of standalone interactive HTML apps for teaching.

## Gallery

**[View all apps →](https://davidmstraub.github.io/teaching-apps/)**

## Current Apps

- **Bézier Curve Editor** - Piecewise Bézier curves with draggable control points, degree and continuity (C0/G1/C1/C2) options

## Usage

All apps are standalone HTML files with no build step. Open any app directly or browse the gallery at `index.html`.
